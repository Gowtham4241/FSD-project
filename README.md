FusionFit – Custom T-Shirt Store 👕✨

FusionFit is a modern and interactive custom T-shirt shopping website built using HTML, CSS, and JavaScript.
The project features stylish UI animations, category-based product collections, responsive design, and an engaging shopping experience for users.

🚀 Features
🎨 Modern futuristic UI design
🦸 Superhero collection
🍥 Anime collection
⚽ Sports collection
🛒 Add to Cart functionality
✨ Hover animations and transitions
📱 Fully responsive design
🔄 Dynamic page navigation using JavaScript
💡 Interactive product cards with front/back image effects
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Font Awesome Icons
Google Fonts
